#EduSpace  — Платформа внеучебной деятельности

Информационная система управления школьными ресурсами (библиотека, кружки и секции)

## 🚀 Основные возможности

### 🎭 Для учащихся
- **Интерактивный дашборд:** Современный дизайн с Hero-слайдером и плавными анимациями.
- **Онлайн-запись:** Удобные формы для записи в секции и кружки.
- **Умная библиотека:** Быстрая аренда книг через систему сканирования QR-кодов.
- **Адаптивность:** Полная поддержка мобильных устройств (смартфоны, планшеты).

### 👨‍🏫 Для преподавателей (Админ-панель)
- **Управление направлениями:** Создание и удаление спортивных секций и творческих кружков.
- **Электронный журнал:** Просмотр списка всех зарегистрированных учащихся в реальном времени.
- **История библиотеки:** Контроль выдачи и возврата книг с возможностью смены статуса.
- **Экспорт данных:** Выгрузка стилизованных отчетов в Excel (XLSX) с поддержкой фильтров.
- **Генератор QR:** Создание QR-кодов для библиотечного фонда.

## 🛠 Технологический стек
- **Frontend:** React + Vite
- **Backend/DB:** Supabase (PostgreSQL + Realtime)
- **Styling:** Modern Vanilla CSS (Design system, Flex/Grid)
- **Integration:** HTML5 QR Scanner, XLSX-js-style

## 📦 Установка и запуск

### 1. Предварительные требования
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18.x или выше).

### 2. Клонирование и установка
Скачайте проект и установите необходимые зависимости:

```bash
# Установка зависимостей
npm install
```

### 3. Настройка базы данных
Создайте файл `.env` в корне проекта (если используете переменные окружения) или настройте параметры в `src/lib/supabaseClient.js`:
- `SUPABASE_URL`
- `SUPABASE_KEY`

### 4. Запуск проекта
Для запуска в режиме разработки:

```bash
npm run dev
```
После запуска проект будет доступен по адресу: `http://localhost:5173`

## 📊 Зависимости
Проект использует следующие ключевые библиотеки:
- `react-router-dom` — Навигация между страницами.
- `@supabase/supabase-js` — Взаимодействие с базой данных.
- `html5-qrcode` — Сканирование кодов через камеру.
- `qrcode.react` — Генерация QR-кодов.
- `xlsx-js-style` — Экспорт данных в Excel с сохранением стилей.

## 📱 Адаптивная верстка
Проект оптимизирован под:
- **Desktop:** Широкоформатные мониторы и ноутбуки.
- **Tablets:** Портретная и ландшафтная ориентация.
- **Mobile:** Узкие экраны (iPhone SE/12 mini и др.) без горизонтальной прокрутки.
